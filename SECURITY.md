# Security Policy

## Supported Versions

The latest stable release receives security fixes.

| Version | Supported |
| ------- | --------- |
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Report vulnerabilities using one of these channels:

- GitHub Security Advisories (preferred):
	`https://github.com/rs-mini-rgb/RNG_Shunt300-ls/security/advisories/new`
- Email: `richsantos78@gmail.com`

Include:

- A clear description of the issue and impact.
- Steps to reproduce.
- Affected version/commit.
- Any proof-of-concept details.

Response targets:

- Initial response: within 72 hours.
- Status updates: at least every 7 days until resolution.

If the report is valid, we will coordinate a fix and responsible disclosure,
then publish release notes and mitigation guidance.
